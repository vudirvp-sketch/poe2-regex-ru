## Оставшиеся задачи

### 🔴 P0 — Критично

1. **Упростить prefix anchoring** — cross-mod FP не существует в игре. Prefix anchoring всё ещё полезен для disambiguation внутри одного мода, но его можно значительно упростить.
2. **Обновить ETL compute-regex** — заменить "использ" на "зарядов" для планшетов (verified in-game)
3. **Re-run ETL** — `pnpm etl` чтобы перегенерировать JSON с обновлённой моделью

### 🟡 P1 — Важно

4. **Фаза 8: Cross-family FP reduction** — использовать блок-модель для более точного FP анализа
5. **Интеграция `matchPoE2RegexItem` в Oracle** — Oracle должен учитывать блок-модель
6. **Удалить deprecated `getItemSearchText()`** — заменить все вызовы на `getItemSearchBlocks()`
7. **jewel-corrupted** — avg regex len = 7.4, добавить в STRICT_CATEGORIES

### 🟢 P2 — Желательно

8. **Фазы 3-8 оптимизатора** — DP на Trie, диалектные оптимизации, итеративный цикл
9. **Виртуализация списков** (belt 298, ring 366, amulet 427)
10. **Multi-line mod handling** — Разрушительный: каждая под-строка = отдельный блок в ETL
