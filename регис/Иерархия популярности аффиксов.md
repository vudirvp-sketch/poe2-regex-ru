# Иерархия популярности аффиксов PoE2 (мета Dawn of the Hunt / 0.3+)

> Источники: Maxroll.gg, Mobalytics.gg, poe2db.tw, Reddit r/PathOfExile2, U4GM, MTMMO

---

## ПУТЕВЫЕ КАМНИ (Waystones)

Префиксы = лут (хорошо), Суффиксы = опасность (но дают Waystone Drop Chance).

### Тир S (ищут ВСЕГДА)
- Количество найденных предметов (Quantity) — умножает ВСЁ
- Редкость найденных предметов (Rarity) — MF
- Размер групп монстров (Pack Size) — больше мобов = больше лута
- Количество групп монстров — прямое увеличение плотности

### Тир A (хорошие префиксы)
- Волшебные монстры / Редкие монстры — лучший лут
- Опыт — для левелинга
- Волшебные/редкие сундуки

### Тир B (нишевые)
- Золото — только для голд-фарма
- Дополнительные паки определённого типа мобов — ситуативно

### Суффиксы (опасные, но нужные для фарма камней)
| Опасность | Примеры модов | Waystone Drop |
|-----------|---------------|---------------|
| 🟡 Низкая | Доп. стихийный урон мобам, порог оглушения | 75-95% |
| 🟠 Средняя | +Урон мобов, +здоровье мобов, пробитие ресов мобами | 90-125% |
| 🔴 Высокая | -60% скорость восстановления, -50% перезарядка, -заряды флаконов | 110-125% |

## ПЛИТКИ ПРЕДТЕЧ (Tablets)

### Тир S (ВСЕГДА искать)
- % увеличения количества предметов в картах (Quantity) — самый ценный, умножает ВСЁ в радиусе
- % увеличения редкости (Rarity)

### Тир A
- Шанс дополнительной механики (доп. Бездна/Ритуал/Экспедиция)
- Опыт в картах
- Редкие монстры в картах

### Тир S суффиксы (зависят от типа)
| Тип планшета | Самый ценный суффикс |
|-------------|---------------------|
| Ритуал | Дополнительный переролл у алтарей (самый дорогой мод) |
| Ритуал | Шанс оменов у алтарей |
| Бездна | Больше осколков Бездны |
| Экспедиция | Больше взрывчатки |

### Тир C (не ценятся)
- Моды на конкретный тип мобов, моды на сложность мобов

## КОЛЬЦА (Rings)

### Тир S (ВСЕГДА искать)
- +1/+2/+3 к уровню камней умений — самый ценный аффикс
- +Дух (Spirit) — критический ресурс
- +Ко всем стихийным сопротивлениям — главный источник ресов
- +Энергетический щит / +% ES

### Тир A
- +Атрибуты (30+)
- Добавленный стихийный урон к атакам
- +% к урону стихией
- Скорость сотворения чар / атаки
- Регенерация маны

### Тир B
- Хаос-сопротивление, MF, +Максимум маны

### Тир C (мало кому нужны)
- Похищение жизни/маны, радиус обзора, шипы, порог оглушения

### Ценовые ориентиры (Div)
- +2/+3 Skills + Spirit + All Res = 10-35+ Div

## АМУЛЕТЫ (Amulets)

### Тир S (АБСОЛЮТНЫЙ ПРИОРИТЕТ)
- +2/+3 к уровню камней умений (снаряды/ближний бой/чары) — самый желанный мод
- +Дух — один из немногих источников
- +% ES / +Max ES
- +Ко всем сопротивлениям / +Ко всем атрибутам

### Тир A
- +Максимум маны / +% маны
- Глобальная защита
- Скорость сотворения чар
- Регенерация маны

### Тир B
- Хаос-сопротивление, MF, добавленный урон

### Тир C
- Модификаторы приспешников (только миньон-билды), радиус обзора

### Базы
- Solar Amulet — +3 Projectile (implicit Spirit)
- Stellar Amulet — атрибуты
- Azure Amulet — ES/Mana

## ПОЯСА (Belts)

### Тир S (АБСОЛЮТНЫЙ ПРИОРИТЕТ)
- +Максимум здоровья (150+) — жизнь дефицитна, пояс = главный источник
- +Ко всем стихийным сопротивлениям — «несёт половину всех ресов»
- Скорость восстановления здоровья флаконом — синергия с Eternal Youth

### Тир A
- Отдельные сопротивления (30-48% к одной стихии)
- +Атрибуты (сила/ловкость/интеллект)
- Скорость восстановления маны флаконом
- Хаос-сопротивление
- Броня

### Тир B
- Длительность оберегов, заряды флаконов

### Тир C
- Шипы, радиус обзора, моды приспешников

### Ценовые ориентиры
- 150+ Life + 30%+ All Res + Flask Recovery = очень дорогой пояс
