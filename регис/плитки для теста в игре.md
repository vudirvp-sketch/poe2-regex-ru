## Тесты (верифицированы)

### T1 — B2: `"A|B" "!C"` — OR + exclude ✅

```
"к сопротивлению молнии|к сопротивлению холоду" "!хаосу"
```

| Предмет | Почему | Ожидание |
|---------|--------|----------|
| Отвратительное потрясение | +35% молния, без хаоса | ✓ Подсвечивает |
| Унылый фермуар | +12% холод, без хаоса | ✓ Подсвечивает |
| Крутящий горжет | +24% молния, без хаоса | ✓ Подсвечивает |
| Племенной медальон | +34% молния, НО +14% хаос → excluded | ✗ НЕ подсвечивает |

Проверка: OR-match на want + item-wide `!` exclude — `"!A|B"` исключает если найден A ИЛИ B

---

### T2 — B3: `"A|B"` — ни один не найден ✅

```
"к сопротивлению огню|к мудрости"
```

Ожидание: **0 предметов** подсвечено. Ни «огню» ни «мудрости» нет ни на одном предмете.

Проверка: OR не нашёл ни одну альтернативу → ничего не матчит

---

### T3 — C3: `"A" "B"` — AND, один не найден ✅

```
"максимуму здоровья" "к интеллекту"
```

Ожидание: **0 предметов**. Кольцо1 имеет здоровье но не интеллект, Амулет1 имеет интеллект но не здоровье — AND требует ОБА.

Проверка: AND требует оба условия в любом порядке, на любых блоках предмета

---

### T4 — F2: `(N1|N2)%.*suffix` — значение вне диапазона ✅

```
"(2[5-9]|3[0-5])%.*редких монстров"
```

| Предмет | Значение | Попадает в 25-35? | Ожидание | Результат |
|---------|----------|-------------------|----------|-----------|
| Фениксовое побуждение | 27% | 25-29 ✓ | ✓ Подсвечивает | ✅ Совпало |
| Потусторонний ордер | 39% | ВНЕ 25-35 | ✗ НЕ подсвечивает | ✅ Совпало |

Проверка: enumeration НЕ выходит за верхнюю границу — 39 не попадает в 25-35. Результат: **27% подсветило, 39% не подсветило** — паттерн работает корректно.

---

### T5 — H1: `"От N.*suffix"` — dual-number prefix (матчит) ✅

```
"От ([1-9][0-9]).*огня к атакам"
```

| Предмет | Почему | Ожидание |
|---------|--------|----------|
| Расколотый завиток | «от 17 до 30 урона от огня к атакам» — 17 двухзначное, .* до «огня к атакам» | ✓ Подсвечивает |
| Все остальные | Нет «От NN» + «огня к атакам» в одном блоке | ✗ НЕ подсвечивает |

Проверка: prefix «От» привязывает число к первому placeholder, .* внутри одного блока до suffix

---

### T6 — H2: `"От N.*suffix"` — число не подходит ✅

```
"От ([1-9][0-9]).*к атакам"
```

| Предмет | Почему | Ожидание |
|---------|--------|----------|
| Расколотый завиток | «от 17» → 2 цифры, .* до «к атакам» в том же блоке | ✓ Подсвечивает |
| Ненавистное потрясение | «от 5» → 1 цифра, не попадает в [1-9][0-9] | ✗ НЕ подсвечивает |
| Отвратительное потрясение | Нет «От» вообще | ✗ НЕ подсвечивает |

Проверка: число после «От» должно соответствовать паттерну — однозначное не матчит
