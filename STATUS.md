## Итерация 14: Threshold mode + Middle-number + Safe list ✅

### Что сделано

**1. Threshold mode в UI**
- Переключатель `≥Мин` в `CategoryControlPanel` — появляется когда заданы оба min+max
- `useCategoryPage` добавлен `thresholdEnabled` state с URL-синком и профилями
- Компилятор уже поддерживал `threshold=true` — теперь UI его использует
- Effect: RANGE(min,max) компилируется как ≥min только — короче regex, нет FP от диапазонной нотации

**2. Расширен truncated tail safe list**
- Новые записи: `приспешников→приспешник`, `приспешника→приспешник`, `оглушения→оглушен`, `флакона→флакон`, `хаосу→хаос`, `монстров→монстр`
- Требуют in-game верификации (добавлены как кандидаты)

**3. Middle-number паттерны (типы 3 и 9)**
- `getPrefixForSlot` теперь извлекает prefix из `rawTextTemplate` для slot 0, когда `regexPrefix` пуст
- Effect: `"Монстры с N%.*шансом"` вместо `"N%.*шансом"` — дополнительная привязка числа
- Работает на уровне runtime — не требует перезапуска ETL

### Статус паттернов

| # | Паттерн | Пример | Компилятор | In-game |
|---|---------|--------|------------|---------|
| 1 | `хх% бла бла` | `28% увеличение урона` | ✅ anchorEnd=`%` | ✅ |
| 2 | `+хх бла бла` | `+66 к максимуму здоровья` | ✅ `^\+N.*suffix` | ✅ |
| 3 | `бла бла хх бла бла` | `Монстры с 36% шансом...` | ✅ prefix+suffix | ✅ (улучшено) |
| 4 | `бла бла +хх%` | `Редкость предметов: +18%` | ✅ `suffix.*\+N%` | ✅ |
| 5 | `-хх% бла бла` | `-11% максимум сопротивлений` | ✅ `^-N%.*suffix` | ✅ |
| 6 | `+хх% бла бла` | `+35% к сопротивлению молнии` | ✅ `^\+N%.*suffix` | ✅ |
| 7 | `бла бла х` | `Дополнительных снарядов: 2` | ✅ colonAnchor | ✅ |
| 8 | `бла бла хх` | `Осталось зарядов - 10` | ✅ reversed | ✅ |
| 9 | `бла бла хх% бла бла` | `Монстры с 36% шансом` | ✅ prefix+%+suffix | ✅ (улучшено) |

### Ключевые верифицированные факты

1. **`^\+` и `^-`** — якорят к началу блока + матчат знак. Без FP от чисел без знака.
2. **`\+` в reversed-паттернах** — `"Редкость предметов.*\+N%"` работает корректно.
3. **`!` item-wide** — подтверждено: если `!молнии|хаосу` находит «молнии» в ЛЮБОМ блоке предмета — весь предмет исключается.
4. **Threshold mode** — RANGE(min,max) с `threshold=true` → ≥min только. Нет FP от диапазонной нотации, короче regex.
5. **Middle-number prefix** — `getPrefixForSlot` извлекает текст перед ## для типов 3/9. Улучшает привязку числа.
